# 🌱 About EcoSphere

EcoSphere is a next-generation ESG platform designed to make sustainability management intuitive, engaging, and visually immersive.

Instead of presenting ESG information through traditional dashboards, EcoSphere introduces an interactive digital ecosystem where sustainability metrics come to life inside a beautiful forest environment.

Organizations can monitor Environmental, Social, and Governance performance, track organizational growth, visualize sustainability impact, and encourage employee engagement through a premium user experience inspired by nature.

# 🌲 Design Philosophy

EcoSphere is built around the philosophy that sustainability should feel alive rather than static.

Instead of overwhelming users with spreadsheets and charts, the platform visualizes organizational sustainability through a living digital forest.

Every interaction reinforces the idea that stronger ESG practices contribute to a healthier ecosystem, creating a more intuitive and engaging experience for organizations and their employees.

# 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/your-username/ecosphere.git
```

Navigate to the project

```bash
cd ecosphere
```

Install dependencies

```bash
npm install
```

Run development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```

---

# 🔧 Environment Variables

Create a `.env` file in the project root.

```env
VITE_API_URL=http://localhost:5000/api
```

Modify the URL according to your backend deployment.
