## ✨ Overview

EcoSphere reimagines ESG management as an immersive digital ecosystem.

Instead of navigating traditional dashboards, users enter a living forest where sustainability metrics become interactive visual experiences.

Every interaction has been designed to make ESG data intuitive, engaging and memorable.

## 📊 Backend

The frontend is designed around a scalable service architecture.

```

AuthService

ESGService

DepartmentService

ReportService

LeaderboardService

RewardService

ChallengeService

NotificationService
```

API calls are abstracted from UI components for maintainability.

---

## 🌲 Design Philosophy

EcoSphere was inspired by the idea that sustainability should feel alive.

Instead of conventional dashboards, organizations explore a digital forest where:

🌳 The Mother Tree represents organizational ESG health.

🌿 Branches represent ESG pillars.

🍃 Growth symbolizes sustainability.

✨ Particles and environmental effects respond to ESG performance.
